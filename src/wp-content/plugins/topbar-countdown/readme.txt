Add a banner on the top of screen with countdown clock and custom message.

== Description ==

This plugin will add Topbar Countdown on the top of your website that help you launch an offer or a promotions.

### Featured ###
 * 2 Countdown type: Fixed timer for all visits and Start the countdown when first visits.
 * Custom page or post type to show.
 * Custom message.
 * Custom countdown box title.
 * Custom Input coupon box.
 * 2 Countdown box title: Vertical and Horizontal.
 * Advanced color settings.


== Installation ==
1. Upload the entire `topbar-countdown.zip` folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
