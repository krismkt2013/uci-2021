Sticky Topbar adds an area to the top of your website and lets you customize it from the WordPress setting options page. The easiest way to add a topbar to your website.

== Description ==

Sticky Topbar plugin lets user add an area at top of the website which is sticky. This plugin compatible with any WordPress themes. User can add social media URLs, call-to-action text with button and countdown timer. Fully responsive.

There are many ways to add a topbar in WordPress website but flexibility of Sticky Topbar plugin helps user to do much more than any other topbar plugin. Using the plugin the website owner can promote his/her book, event ticket or any other landing page with a countdown timer.

= Features =
* Add call-to-action text
* Add call-to-action button
* Social media icons
* Color changes capability
* Dismiss button
* Countdown timer. [See Demo](https://rainastudio.com/demo/sticky-genesis-topbar/)


You will be able to add your brand's social URL of following sites:
* Facebook
* Twitter
* Linkedin
* Pinterest
* Instagram
* YouTube
* Dribbble
* Medium
* Tumblr
* Vimeo
* Mobile/Cell Number

= Compatibility =

Sticky Topbar is compatible with any WordPress theme.

== Installation ==

1. Upload the entire sticky-topbar folder to the /wp-content/plugins/ directory (or install directly by searching in Plugins > Add New)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In Settings menu you'll see submenu "Sticky Topbar", add or modify content for your topbar from the meta fields.
